# GALLERYX

A sleek, dark-themed photo gallery web app with a **Maybe Later** feature — save photos you want to revisit with one click.

## Features

- 📸 Responsive photo grid with smooth hover effects
- 🔖 **Maybe Later** — bookmark any photo to your personal saved list
- 🕰️ Dedicated **Maybe Later** tab to review and manage saved photos
- 💾 Saved items persist across browser sessions via `localStorage`
- 🌑 Dark UI with clean, minimal design

## Getting Started

Open `index.html` directly in your browser — no build step or server required.

```
open index.html
```

## Project Structure

```
index.html   Main markup
style.css    Styles
app.js       Gallery logic & Maybe Later feature
```
